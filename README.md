🔐 EntroX – Password Strength Analyzer

In today’s digital age, passwords protect our most personal information from social media accounts to online banking. However, many users still create weak and predictable passwords, making them easy targets for cyber attacks. EntroX was developed to help users understand how secure their passwords truly are.

This project analyzes passwords in real time by checking their length, use of uppercase and lowercase letters, numbers, and special characters. Based on these factors, it calculates a strength score and visually displays whether the password is weak, moderate, or strong, along with suggestions for improvement.

EntroX is not just a password checker it promotes cybersecurity awareness by encouraging users to build stronger digital habits and protect their online identity more effectively.
